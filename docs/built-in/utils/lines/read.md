# read
<!--endTocHeader-->

```
Read a text file and return a jsonStrList

Usage:
  zaruba lines read <strFileName> [flags]

Examples:

> cat fruits.txt
🍊
🍓
🍇
> zaruba lines read fruits.txt
["🍊","🍓","🍇"] 


Flags:
  -h, --help   help for read

```

# get
<!--endTocHeader-->

```
Get value of nested JsonMap or JsonList

Usage:
  zaruba json get <jsonAny> <key> [flags]

Aliases:
  get, write

Examples:

> zaruba json get '{"characters": [{"name": "doraemon"}, {"name": "nobita"}]}' 'characters.[0].name'
"doraemon"


Flags:
  -h, --help   help for get

```

# json
<!--endTocHeader-->

```
Json manipulation utilities

Usage:
  zaruba json [command]

Available Commands:
  get         Get value of nested JsonMap or JsonList
  print       Print json
  set         Set value of nested JsonMap or JsonList

Flags:
  -h, --help   help for json

Use "zaruba json [command] --help" for more information about a command.
```

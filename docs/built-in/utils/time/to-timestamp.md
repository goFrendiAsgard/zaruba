# toTimestamp
<!--endTocHeader-->

```
Encode a string with time algorithm

Usage:
  zaruba time toTimestamp <RFC-3339> [flags]

Examples:

> zaruba time toTimestamp '2023-01-01T13:01:01.345Z'
1672578061

> zaruba time toTimestamp '2023-01-01T20:01:01+07:00'
1672578061


Flags:
  -h, --help   help for toTimestamp

```

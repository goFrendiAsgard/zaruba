# read
<!--endTocHeader-->

```
Read YAML file as JSON map or list

Usage:
  zaruba yaml read <yamlFileName> [flags]

Examples:

> cat book.yaml
id: 1
title: Doraemon

> zaruba yaml read book.yaml
{"id":1,"title":"Doraemon"}


Flags:
  -h, --help   help for read

```

# list
<!--endTocHeader-->

```
List files/folders in a path

Usage:
  zaruba file list <strDirPath> [flags]

Examples:

> ls myDir
a.txt   b.txt

> zaruba file list myDir
a.txt
b.txt


Flags:
  -h, --help   help for list

```

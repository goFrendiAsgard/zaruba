# copy
<!--endTocHeader-->

```
Copy files/folders recursively

Usage:
  zaruba file copy <strSourcePath> <strDestinationPath> [flags]

Examples:

> ls myDir
main.py    module

> zaruba file copy myDir otherDir

> ls otherDir
main.py    module


Flags:
  -h, --help   help for copy

```

# merge
<!--endTocHeader-->

```
Merge multiple jsonMaps

Usage:
  zaruba map merge <jsonMap> <otherJsonMaps...> [flags]

Examples:

> zaruba map merge '{"server": "localhost", "port": 3306}' '{"protocol": "mysql"}'
{"server": "localhost", "port": 3306, "protocol": "mysql"}


Flags:
  -h, --help   help for merge

```

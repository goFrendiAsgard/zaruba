# getKeys
<!--endTocHeader-->

```
Return a jsonStringList containing all keys in a jsonMap

Usage:
  zaruba map getKeys <jsonMap> [flags]

Examples:

> zaruba map getKeys '{"server": "localhost", "port": 3306}' server
["server", "port"]


Flags:
  -h, --help   help for getKeys

```

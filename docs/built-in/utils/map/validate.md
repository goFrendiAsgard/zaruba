# validate
<!--endTocHeader-->

```
Check whether jsonMap is valid JSON map or not

Usage:
  zaruba map validate <jsonMap> [flags]

Examples:

> zaruba map validate '{"server": "localhost", "port": 3306}'
1

> zaruba map validate 'not a map'
0


Flags:
  -h, --help   help for validate

```

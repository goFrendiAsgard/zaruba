# set
<!--endTocHeader-->

```
Set a value of a jsonMap on a particular key

Usage:
  zaruba map set <jsonMap> <key> <value> [<otherKey> <otherValue>...] [flags]

Examples:

> zaruba list set '{"server": "localhost", "port": 3306}' server stalchmst.com
{"server": "stalchmst.com", "port": 3306}


Flags:
  -h, --help   help for set

```

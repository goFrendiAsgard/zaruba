# join
<!--endTocHeader-->

```
Transform a jsonList into single string

Usage:
  zaruba list join <jsonList> [separator] [flags]

Examples:

> zaruba list join '["🍊","🍓","🍇"]' ";"
🍊;🍓;🍇


Flags:
  -h, --help   help for join

```

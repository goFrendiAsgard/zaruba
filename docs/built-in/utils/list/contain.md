# contain
<!--endTocHeader-->

```
Find out whether a jsonList contains an element or not

Usage:
  zaruba list contain <jsonList> <strElement> [flags]

Examples:

> zaruba list contain '["🍊","🍓","🍇"]' 🍓
1

> zaruba list contain '["🍊","🍓","🍇"]' 🍕
0


Flags:
  -h, --help   help for contain

```

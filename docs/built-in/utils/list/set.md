# set
<!--endTocHeader-->

```
Replace jsonList element at a particular index with a value

Usage:
  zaruba list set <jsonList> <index> <value> [flags]

Examples:

> zaruba list set '["🍊", "🍓", "🍇"]' 1 🍕
["🍊","🍕","🍇"]


Flags:
  -h, --help   help for set

```

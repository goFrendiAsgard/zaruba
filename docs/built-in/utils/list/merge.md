# merge
<!--endTocHeader-->

```
Merge JSON lists

Usage:
  zaruba list merge <jsonList> <jsonListOther...> [flags]

Examples:

> zaruba list merge '["🍊","🍓","🍇"]' '["🍎","🍏"]' '["🍕"]'
'["🍊","🍓","🍇","🍎","🍏","🍕"]'


Flags:
  -h, --help   help for merge

```
